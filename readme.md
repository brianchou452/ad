# Dcard Backend Intern Assignment

## Introduction

使用 Golang 設計並且實作一個簡化的廣告投放服務,該服務應該有兩個 API,一個用於產生廣告,一個用於列出廣告。每個廣告都有它出現的條件(例如跟據使用者的年齡),產生廣告的 API 用來產生與設定條件。投放廣告的 API 就要跟據條件列出符合使用條件的廣告
